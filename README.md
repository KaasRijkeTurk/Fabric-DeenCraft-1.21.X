# DeenCraft (Fabric 1.21.10)

DeenCraft is a Fabric mod that adds Islamic-inspired items and blocks to Minecraft, focused on calm and immersion.

This mod is still work in progress.

## Content

### Blocks
- Prayer Mat (red, blue, green)

### Items
- Wooden Tasbih
- Iron Tasbih
- Dates
- Big Date
- Astrolabe

Currently, the prayer mats are functional and the items have crafting recipes.  
Other items are added but still being worked on.
